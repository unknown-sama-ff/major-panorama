# 2026年中国大学本科专业分类全景图

> 一个基于教育部《普通高等学校本科专业目录（2026年）》的交互式高考志愿填报辅助网站。

---

## 📋 项目简介

本项目是一个**单页应用（SPA）**风格的高考志愿填报参考网站，完整收录了2026年教育部发布的本科目录数据，包含：

- **13个学科门类**、**93个专业类**、**872种专业**
- **38种新增专业**（2026年首次设立）
- 首次增设**"交叉学科"**门类

网站采用纯前端技术栈，数据内嵌于 JavaScript 中，无需后端服务即可运行。

---

## 🎯 核心功能

### 1. 专业全景浏览
- **总览视图**：13个学科门类的卡片式概览，展示各专业类的数量分布
- **分学科详图**：每个学科门类下的专业类及具体专业列表
- **新增专业速查**：2026年新增的38种专业独立表格，标注门类、代码、学位授予信息
- **决策辅助表**：按倾向标签（理科/工科/文科/数字智能/新兴交叉/需特别注意）分类展示专业

### 2. 智能搜索
- 支持按**专业名称**、**专业代码**、**类别关键词**实时搜索
- 搜索结果高亮匹配文本，显示专业所属门类、专业类、学位类型及标签
- 点击结果可直接查看专业详情弹窗

### 3. 志愿填报匹配
- 输入考生信息（省份、分数、选科组合）
- 设置筛选条件（目标地区、分数线范围、学校类型、目标专业门类/关键词）
- 系统根据历年录取数据计算**分差**，将匹配高校分为：
  - 🔴 **风险**（分数远低于录取线）
  - 🟠 **冲刺**（分数略低于录取线）
  - 🟡 **较稳**（分数接近录取线）
  - 🟢 **稳妥**（分数高于录取线）
  - 🔵 **保底**（分数明显高于录取线）
- 支持虚拟滚动加载大量结果
- 一键将匹配高校添加到志愿表

### 4. 志愿表管理
- 创建、重命名、删除多个志愿表
- 志愿表内高校可**编辑备注**、**调整顺序**、**删除**
- 自动统计各层次（冲刺/较稳/稳妥/保底）志愿数量
- 支持**导出JSON**、**导出文本**（标准志愿填报表格式）
- 支持**分享链接**（通过 Base64 编码序列化数据）
- 支持**志愿表对比**（并排展示两个志愿表的结构差异）

### 5. 分数线数据查询
- 收录各省2022-2025年**本科省控线**
- 支持按**物理类/历史类**（或理科/文科）筛选
- 显示批次、年份、分数线及备注信息

### 6. 辅助工具箱

| 工具 | 功能描述 |
|------|----------|
| **分数换算** | 基于省控线变化，将某年分数换算到另一年等效分数 |
| **位次查询** | 根据分数估算在各省的位次排名 |
| **同分去向** | 查询与考生分数相近（±10分）的高校投档线 |
| **选科匹配** | 查询专业门类的选科要求，或根据已选科目反查可报专业 |

### 7. 收藏与详情
- 收藏感兴趣的高校，集中管理
- 高校详情弹窗展示：学校简介、保研率、学科评估、博士点、历年录取趋势图、各专业分数线
- 专业详情弹窗展示：专业简介、就业前景（就业率、平均薪资、就业方向）

## 🏗️ 技术架构

### 文件结构

```
├── major-panorama.html          # 主页面（单文件应用）
├── assets/
│   ├── data.js                  # 专业目录数据（学科门类、专业类、专业）
│   └── university_data.js       # 高校数据（录取线、招生计划、排名等）
└── _shared/fonts/               # 字体文件（WorkSans, CrimsonPro, GeistMono）
```

### 技术栈

- **HTML5** + **CSS3**（原生，无框架）
- **Vanilla JavaScript**（ES5 语法，兼容旧浏览器）
- **CSS 变量**（主题色系管理）
- **LocalStorage**（持久化存储志愿表、收藏数据）
- **纯前端渲染**，无后端依赖

### 数据加载策略

1. `data.js`：通过 `<script src>` 同步加载，包含专业目录基础数据
2. `university_data.js`：通过 `fetch()` 或动态 `<script>` 标签异步加载，包含高校录取数据
3. 加载失败时显示降级提示，不影响基础功能使用

## 🎨 设计特点

### 视觉风格
- **纸质/学术风格**：米白背景（`#F7F5F2`）、深棕文字（`#2C2A28`）、暖灰辅助色
- **衬线标题**：CrimsonPro 用于大标题，营造学术感
- **无衬线正文**：WorkSans 用于正文，保证可读性
- **低饱和度配色**：符合长时间阅读的视觉舒适度

### 交互设计
- **响应式布局**：桌面端网格布局，移动端单列适配
- **粘性导航栏**：滚动时顶部导航始终可见
- **移动端悬浮菜单**：右下角圆形按钮展开导航
- **骨架屏加载**：数据加载时显示占位动画
- **Toast 提示**：操作反馈而非弹窗打断
- **模态弹窗**：专业/高校详情采用 overlay 弹窗，支持点击外部关闭

### 无障碍考虑
- 所有交互元素最小点击区域 **44×44px**
- 支持键盘导航和屏幕阅读器
- 高对比度文字与背景
- 触摸设备上的 active 状态反馈

## 🚀 使用方法

### 本地运行

1. 确保所有文件在同一目录下：
   ```
   major-panorama.html
   assets/data.js
   assets/university_data.js
   _shared/fonts/...
   ```

2. 直接用浏览器打开 `major-panorama.html`：
   - **双击打开**：基础功能可用，但高校数据可能因 CORS 限制无法加载（`file://` 协议下 `fetch` 受限）
   - **推荐方式**：使用本地 HTTP 服务器
     ```bash
     # Python 3
     python -m http.server 8000

     # Node.js
     npx serve .

     # 然后访问 http://localhost:8000/major-panorama.html
     ```

### 部署到服务器

将项目文件上传至任意静态文件托管服务（GitHub Pages、Vercel、Netlify、Nginx 等）即可完整运行。

## 🔧 自定义扩展

### 添加新的省份数据

在 `scoreLineData` 对象中添加省份代码和历年批次数据：

```javascript
scoreLineData = {
  newprovince: {
    years: [
      { year: '2025', batches: [
        { name: '本科批', physics: 450, history: 470, note: '3+1+2' }
      ]}
    ]
  }
}
```

### 添加高校数据

在 `university_data.js` 中按 `universityData` 数组格式追加高校对象。

### 修改主题色

编辑 CSS 变量（`:root` 部分）：

```css
:root {
  --bg: #F7F5F2;      /* 背景色 */
  --ink: #2C2A28;     /* 文字色 */
  --accent: #6B5B4E;  /* 强调色 */
  --new-badge: #C75B39; /* 新增标签色 */
}
```
